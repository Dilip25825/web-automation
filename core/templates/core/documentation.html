सरल उपयोग गाइड

Coupon कैसे इस्तेमाल करें?

Coupon प्राप्त करने, client को भेजने और Excel में apply करने की आसान प्रक्रिया।

Portal से coupon copy करें

1

Coupons खोलें

Login करने के बाद navbar या Main Hub से Coupons पर जाएं।

2

Available चुनें

Available filter में client की जरूरत के अनुसार discount वाला coupon देखें।

3

Copy icon दबाएं

Coupon code के पास Copy icon दबाएं। Coupon copy होकर अपने-आप Reserved हो जाएगा।

4

Client को भेजें

Copied code WhatsApp, message या अन्य सुरक्षित माध्यम से client को भेज दें।

ध्यान दें: एक coupon copy होने के बाद Reserved दिखता है। नया coupon देना हो तो हमेशा Available list से चुनें।

Client Excel में coupon कैसे लगाए?

  1. 1
    Excel software खोलें

    Client अपनी registered Excel file/software खोले और User details load होने दे।

  2. 2
    Ribbon में Happy Face icon दबाएं

    ऊपर ribbon में Coupon/Happy Face वाला button दिखाई देगा। उस पर click करें।

  3. 3
    Coupon code paste करें

    Input box खुलेगा। मिला हुआ पूरा coupon code paste करें और OK दबाएं। आगे-पीछे extra space न रखें।

  4. 4
    Success message देखें

    Coupon सही होने पर Coupon Applied का message और पुराना amount, discount तथा नया amount दिखाई देगा।

  5. 5
    नया amount verify करें

    Payment से पहले payable amount कम हुआ है या नहीं, जरूर check करें। एक user coupon केवल एक बार इस्तेमाल कर सकता है।

Error आए तो क्या करें?

Coupon नहीं मिला

Code दोबारा copy/paste करें और कोई character छूटा है या नहीं देखें।

Coupon inactive या used है

पुराना coupon दोबारा apply नहीं होगा। Coupon देने वाले user से संपर्क करें।

User पहले coupon ले चुका है

एक UserInfo record पर दूसरा coupon apply नहीं किया जा सकता।

Amount से ज्यादा discount

छोटे discount वाला coupon लें या payable amount verify करें।

Admin याद रखने योग्य बातें

  • Add Coupon से single और Bulk Generate से एक साथ कई coupons बनाएं।

  • Active में Available और Reserved दोनों आते हैं; Available में केवल अभी तक न दिए गए coupons आते हैं।
  • Reserved coupon client को दिया गया है लेकिन apply नहीं हुआ। Used/Inactive coupon apply हो चुका है या manually inactive है।

  • Regular users केवल View और Copy कर सकते हैं; Add, Bulk Generate, Edit और Delete backend से भी blocked हैं।
  • कोई bulk delete/truncate करने से पहले database backup जरूर लें।
