Common problems

Application upload fail ho to

  • Invalid format: Official template aur accepted file type check karein.
  • Missing information: Mandatory member, loan, crop aur land details verify karein.
  • Duplicate application: Member/reference ID se existing record check karein.
  • Validation error: Error report ke row aur column ke anusaar data sudharein.
